## capsbrain: 
CapsuleNet(DigitCaps and MatrixCaps) library for training, evaluation and prediction

* Dynamic Routing Between Capsules [[Hinton, 2017 (pdf)](https://arxiv.org/pdf/1710.09829.pdf)]
